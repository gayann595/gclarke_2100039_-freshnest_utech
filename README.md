# gclarke_2100039_-freshnest_utech
FreshNest is a multi-page e-commerce website built with vanilla HTML5, CSS3, and JavaScript. It combines a bakery, grocery store, and laundry service into one accessible platform designed for all users, including persons with physical, visual, hearing, and cognitive challenges. Features include user login and registration with form validation, a dynamic shopping cart with tax and discount calculations, product filtering, and a fully responsive design for both mobile and desktop.
